## ?? If Something Goes Wrong

### Common Issues & Fixes:

**Button doesn't show:**
```
1. Check: ui_src folder copied?
2. Check: mod.json has UIScript field?
3. Fix: Rebuild and recopy files
```

**Universal zones missing:**
```
1. Check: Game logs for errors
2. Wait: Give it 10 seconds after load
3. Fix: Restart game
```

**Settings not working:**
```
1. Check: Mod enabled in mod manager?
2. Try: Change setting and restart
3. Fix: Clear mod settings and reset
```
